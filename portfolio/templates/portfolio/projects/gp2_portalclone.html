GP2 Portal Clone

A portal clone focused on the graphics / portal shader in c++ & HLSL

This project was made for the course Graphics Programming 21 at Howest DAE. I chose to make a portal clone because I wanted to learn more about the portal shader and how it works. The result isn't perfect as I struggled with getting the 'reflection' / see through direction 100% correct. The overlord engine (engine provided by teacher) was used for this project.
